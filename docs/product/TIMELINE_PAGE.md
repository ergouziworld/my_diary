# 时间线页

时间线页以纵向流的形式展示带日期的事件。

已实现的元素：
- 页面标题
- 筛选标签
- 按时间分组的时间线卡片
- 表示事件语气或类型的状态标签

代码位置：
- `src/app/timeline/page.tsx`

设计目标：
- 展示有顺序的生活事件
- 让用户快速扫一眼当天发生了什么
- 支持按事件类型或分类筛选

当前状态：
- 页面布局已经存在
- 时间线数据是静态示例
- 筛选目前只是装饰性的

页面里显示的数据形状：
- `time`
- `title`
- `tag`
- `text`
- `tone`

这个页面和 schema 里的 `TimelineEvent` 概念直接对应，只是后端列表和筛选逻辑还没有接通。
