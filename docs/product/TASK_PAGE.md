# 任务页

任务页是应用里的任务管理界面。

已实现的元素：
- 页面标题
- 任务列表卡片
- 优先级和状态徽标
- AI 建议面板

代码位置：
- `src/app/tasks/page.tsx`

数据模型对应：
- `Task`
- `TaskLink`
- `TaskStatus`
- `PriorityLevel`

设计目标：
- 查看从日记中抽取出来的任务
- 跟踪条目里的待办事项
- 按紧急程度和状态排序

当前状态：
- 视觉壳已经存在
- 任务是静态的
- AI 提示是解释性内容，还没有实时生成

页面里显示的数据形状：
- `title`
- `due`
- `priority`
- `status`

这个页面后续会成为从日记条目抽取任务，以及手工补充任务的复核和分拣界面。
