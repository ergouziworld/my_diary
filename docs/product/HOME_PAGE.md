# 首页

首页是一个看板式的落地页。

已实现的元素：
- 顶部标签行
- 汇总英雄区
- 指标卡片
- 快速输入面板
- 最近条目列表
- 快速操作面板

代码位置：
- `src/app/page.tsx`

当前状态：
- 页面视觉上已经完成成一个壳
- 数据是静态示例
- 页面在传达产品的核心闭环：记录、分析、回顾

这个页面想表达的是：
- 这是一个日常指挥中心，而不只是首页
- 记录入口应该足够快
- AI 操作应该在第一屏就能触达
- 最近内容应该在导航之前先可见

代码里的重要 UI 组件：
- `MetricCard`
- `Panel`
- `Pill`
- `SectionHeader`

首页是整个日记工作区的最高层概览，后续接入实时数据后，最适合承载日常状态。
