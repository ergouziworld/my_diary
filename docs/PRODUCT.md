# 产品

`my_diary` 是一个个人日记工作区，围绕结构化条目、AI 辅助分析和紧凑的回顾页面展开。

产品闭环是：
- 快速记录一条内容
- 在需要时分析这条内容
- 派生出任务、情绪、时间线事件和工作事项
- 在专门页面里回看结果

主要产品对象：
- 条目
- 附件
- 分析结果
- 任务
- 情绪记录
- 时间线事件
- 工作事项
- 对话会话
- 相册条目

当前可见的页面：
- 首页看板
- 时间线
- 任务
- 情绪
- 大模型聊天
- 相册
- 财务
- 设置

产品原则：
- 记录要快
- 以日记正文为唯一事实来源
- 先从原始内容派生结构化对象，不要求用户先手工打标签
- AI 主要用于回顾和拆解，而不只是聊天
