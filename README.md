# My Diary

一个本地优先的个人生活收集箱原型，目标是把日记、照片、任务、记账和 AI 整理放到同一个入口里。

## 现在能做什么

- 随手输入一段文字并本地保存
- 上传图片，压缩后保存在浏览器 IndexedDB
- 自动用本地规则提取任务、账单、情绪和标签
- 查看今天、时间线、任务、账本和最近分析
- 导出/导入 JSON 备份
- 预留 Gemini 和 OpenRouter API 配置入口

## 怎么运行

这是纯静态页面，可以直接打开：

```text
index.html
```

如果你想启用 PWA 离线缓存，建议用本地静态服务打开：

```bash
npx serve .
```

或者：

```bash
python -m http.server 5173
```

然后访问：

```text
http://localhost:5173
```

## 数据存在哪里

数据默认存在当前浏览器的 IndexedDB 中，不上传服务器。

注意：

- 清理浏览器数据可能会删除记录
- 换浏览器或换设备不会自动同步
- 重要数据请定期用右上角导出按钮备份

## AI 接口说明

当前版本优先使用本地规则，已经预留：

- Gemini
- OpenRouter

如果填写 API Key，会尝试远程整理；失败时会自动回退到本地规则。

更适合正式使用的架构是：前端只连自己的后端，API Key 放在后端，不直接暴露给浏览器。
