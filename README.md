# My Diary

本地优先的个人生活收集箱。第一版目标不是分类很细，而是让你先把生活记录下来，再从同一份数据里自动整理出日记、任务、心情、收支和相册。

## 功能

- 本地 IndexedDB 保存文字和图片缩略图
- 保存后立即回读校验，并显示本地记录数
- 今日视图、时间线、任务、心情、收支、相册
- 自动从自然语言里提取任务、金额、心情、标签
- 任务可勾选完成
- JSON 导入和导出备份
- 无后端、无账号、默认不上传任何数据

## 运行

推荐用本地 HTTP 服务：

```bash
python -m http.server 5173 --directory D:\github仓库\my_diary
```

然后打开：

```text
http://127.0.0.1:5173/index.html
```

也可以直接双击 `index.html`，但 PWA 缓存只在 HTTP 环境下启用。

## 本地保存说明

完整数据保存在当前浏览器的 IndexedDB。轻量备份会放在 localStorage，但不包含图片。

如果你清理浏览器数据、换浏览器、换设备，数据不会自动同步。重要记录请定期点左侧“导出”。
